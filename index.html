Tags de Títulos (<h1> a <h6>)

As tags de título são usadas para definir os títulos e subtítulos de uma página. O <h1> é o título mais importante e o <h6> é o menos importante.

Exemplos:

Título Nível 1

Título Nível 2

Título Nível 3

Título Nível 4

Título Nível 5
Título Nível 6

Tag de Parágrafo (<p>)

A tag é usada para definir parágrafos de texto. Cada <p> cria um novo bloco de texto.

Exemplo:

Este é um parágrafo de texto. Ele é usado para organizar o conteúdo em blocos legíveis.

Aqui está outro parágrafo, demonstrando como eles se separam automaticamente.

Tag de Marcação (<mark>)

A tag <mark> é usada para destacar partes do texto, como se tivessem sido marcadas com um marcador de texto.

Exemplo:

Este é um texto com uma parte marcada para destaque.

Voltar

Tag de Texto Pequeno (<small>)

A tag <small> é usada para representar comentários laterais, como direitos autorais ou texto legal. Geralmente, ela renderiza o texto com uma fonte menor.

Exemplo:

Este é um texto pequeno, ideal para notas de rodapé ou informações de copyright.

Voltar

Tag de Texto Itálico (<i>)

A tag <i> é usada para texto que deve ser exibido em itálico. Originalmente era para texto em itálico, mas agora é mais semanticamente usada para voz alternativa ou inclinação técnica, como um termo taxonômico, uma frase de outro idioma, ou uma palavra-chave.

Exemplo:

Este é um texto com uma palavra em itálico.

Voltar

Tag de Texto Sublinhado (<u>)

A tag <u> é usada para representar texto que deve ser exibido com um sublinhado. Originalmente era para sublinhado, mas agora é mais semanticamente usada para um intervalo de texto com uma anotação não textual, como soletrar palavras incorretas.

Exemplo:

Esta frase contém uma palavra sublinhada.

Voltar

Tag de Texto Negrito (<strong>)

A tag <strong> é usada para indicar que o texto tem grande importância. Geralmente, o texto é exibido em negrito.

Exemplo:

É importante lembrar de salvar seu trabalho.

Voltar

Lista Ordenada (<ol>)

A tag <ol> é usada para criar uma lista ordenada de itens. Os itens são numerados automaticamente.

Exemplo:

  1. Primeiro item da lista
  2. Segundo item da lista
  3. Terceiro item da lista

Voltar

Lista Não Ordenada (<ul>)

A tag <ul> é usada para criar uma lista não ordenada de itens. Os itens são marcados com marcadores (geralmente pontos).

Exemplo:

  • Café
  • Chá
  • Leite

Voltar

Item de Lista (<li>)

A tag <li> é usada para definir um item em uma lista (seja ela ordenada <ol> ou não ordenada <ul>).

Exemplo (dentro de uma lista):

  • Item 1
  • Item 2

Voltar

Tag de Link (<a>)

A tag <a> é usada para criar hiperlinks, que conectam uma página a outra ou a uma seção dentro da mesma página.

Exemplo:

Visite o site da DIO para mais informações.

Voltar

Linha Horizontal (<hr>)

A tag <hr> é usada para criar uma linha horizontal, geralmente para separar seções de conteúdo.

Exemplo:

Conteúdo acima da linha.

Conteúdo abaixo da linha.

Voltar

Texto Subscrito (<sub>)

A tag <sub> é usada para exibir texto como subscrito, geralmente para fórmulas químicas ou matemáticas.

Exemplo:

A fórmula da água é H2O.

Voltar

Texto Sobrescrito (<sup>)

A tag <sup> é usada para exibir texto como sobrescrito, útil para expoentes ou notas de rodapé.

Exemplo:

A equação matemática é E = mc2.

Voltar

Citação em Bloco (<blockquote>)

A tag <blockquote> é usada para citar um bloco de texto de outra fonte. Geralmente, ela é recuada.

Exemplo:

"Não vai subir ninguém."

- Capitão Nascimento

Voltar

Tag de Fonte (<font>)

A tag <font> **é uma tag obsoleta em HTML5** e não deve ser usada para estilização. Ela era usada para definir o tamanho, a cor e o tipo da fonte do texto. Para estilização, você deve usar CSS.

Exemplo (apenas para demonstração de como era usada, não recomendado):

Este texto usava a tag <font>.

Voltar

Texto Deletado (<del>)

A tag <del> é usada para representar um intervalo de texto que foi deletado de um documento. Os navegadores geralmente renderizam o texto com uma linha sobre ele (tachado).

Exemplo:

O preço original era R$ 100,00 agora é R$ 75,00.

Voltar

Abreviação (<abbr>)

A tag <abbr> é usada para definir uma abreviação ou um acrônimo. Você pode usar o atributo title para fornecer a descrição completa da abreviação, que é exibida como uma tooltip quando o mouse passa por cima.

Exemplo:

Eu moro no RJ.
